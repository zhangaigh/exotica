Detailed Description

Implementation of distance mesh task map with ROS. Apart from dMesh task map, this task map use exotica and ROS node, msgs, etc. The main improvement is now dMeshROS taking computation, visualisation, and robust into consideration L(d_jl)=K*||p_j-p_l||, K(gain)={PoseGain(kp),ObstacleGain(ko),GoalGain(kg)}.
